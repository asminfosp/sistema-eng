# Checklist de Entrega

## Projeto (GitHub)
- Repositório público
- Estrutura: src/, tests/, docs/
- 10+ commits semânticos
- GitHub Projects (Kanban) com 10 cards
- GitHub Actions (CI) executando Pytest

## Prints obrigatórios
- Kanban no GitHub Projects
- Lista de commits
- GitHub Actions verde (log do pytest)
- Sistema rodando (home e editar)
- Terminal local com pytest passando
