# Sistema de Gerenciamento de Tarefas

Projeto acadêmico da disciplina de Engenharia de Software.

## Objetivo
Desenvolver um sistema web para gerenciamento de tarefas utilizando metodologia ágil.

## Metodologia
Kanban com GitHub Projects.

## Funcionalidades
- CRUD de tarefas
- Visualização Kanban
- Prioridade das tarefas

## Mudança de escopo
Foi adicionada a funcionalidade de prioridade das tarefas.

## Qualidade
Testes automatizados com PyTest e integração contínua com GitHub Actions.
